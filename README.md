# restapi


REST API Client in Java
A lightweight Java application that demonstrates how to interact with external RESTful web services. This project showcases how to send HTTP GET requests, handle network responses, and parse response bodies from public endpoints (like the GitHub API).

🚀 Features
HTTP GET Requests: Connects to standard REST endpoints over HTTPS.

Response Handling: Captures HTTP status codes (e.g., 200 OK) and reads incoming payloads.

Modern vs. Classic Architectures: Contains patterns for both legacy environments (HttpURLConnection) and modern systems (java.net.http.HttpClient).

Robust Stream Management: Implements best practices for auto-closing open network and I/O streams to prevent memory leaks.

🛠️ Tech Stack & Prerequisites
Language: Java (JDK 11 or higher recommended to utilize the native HttpClient)

Target Endpoint: GitHub Users API

IDE: VS Code, IntelliJ IDEA, Eclipse, or any terminal environment.
